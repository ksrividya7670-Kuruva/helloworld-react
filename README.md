# Hello World React App

## Description
This is a simple React application created using Vite.

The application displays the text:

Hello World

## Technologies Used
- React
- Vite
- CSS

## How to Run

1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

3. Open the browser and visit the local URL shown in the terminal.

## Project Structure

src/
 
├── App.js
├── index.js
 

## Output

The application displays a centered "Hello World" heading on the page.
